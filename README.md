# HelloWorld翻译字符破解

## 项目介绍

无限免费字符，由于HelloWorld翻译更新，目前只能破解WhatsApp的翻译字符

## 使用教程

- **1**：启动HelloWorld，选择有道翻译
- **2**：打开电脑的设置，找到代理，代理改为：127.0.0.1  端口改为8080
- **3**：启动运行代码，即可破解成功

## 注意事项
- 需要安装mitmproxy库
- 遇到证书问题需要手动安装证书(安装教程可百度搜索“mitmproxy证书安装”)
